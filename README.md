# MEDIFER Landing — Vite + React + TypeScript

## Requisitos
- Node.js 18+
- npm (o pnpm/yarn)

## Instalar y correr
```bash
npm i
npm run dev
# build de producción
npm run build
npm run preview
```

## Estructura
- `public/logo.png` → Logo integrado (ya copiado del PNG que subiste)
- `src/main.tsx` → Bootstrap y Router
- `src/App.tsx` → Declaración de rutas
- `src/components/MediferLanding.tsx` → Landing completa con animaciones y Smoke Tests
- `vite.config.ts`, `tsconfig.json`, `package.json` → configuración lista
